# Task 7: Fetch and Display Data from Public API

This project uses the JavaScript Fetch API to get user data from a public API and display it on a web page.

## Features

- Fetches user data from: `https://jsonplaceholder.typicode.com/users`
- Displays each user's:
  - Name
  - Email
  - Full address (street, suite, city, zipcode)
- Shows loading status while fetching data
- Handles network / HTTP errors and shows a friendly message
- "Reload Users" button to refetch the data
- Responsive, card-based UI using CSS

## Tech Stack

- HTML
- CSS
- Vanilla JavaScript (Fetch API)
